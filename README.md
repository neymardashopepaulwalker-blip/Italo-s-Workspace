# Italo's Workspace

Welcome to my digital space. This is where I document my journey as a developer, showcasing the projects I have built and the skills I am constantly refining.

## About the Project
This portfolio was designed to be clean, intuitive, and a true reflection of how I approach development. It serves as a hub for my technical projects and a window into my learning path at SESI São Luís.

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript.
- Frameworks/Libraries: React.js, Bootstrap.
- Tools: VS Code, Git, Figma, Notion.
- Database & Backend: MySQL, Express.js, Node.js.

## Featured Projects
- DevLauncher: A tool focused on developer productivity and project management.
- Fitmax: A social-fitness platform featuring gamification and Stripe payment integration. Built to explore full-stack architecture and real-world payment flows.

## The Vibecoding Approach
This portfolio was developed with an AI-assisted workflow. I leveraged tools to accelerate development, experiment with new features, and polish the UI. I see AI not as a replacement for coding, but as a superpower that allows me to bridge the gap between ideas and functional software much faster.
